# Employee-Form
Simple HTML form using Flask with PostgreSQL


  PREREQUISITES:
  
  1) PostgreSQL.
  
  2) Flask.
  
  3) PSQL adapter for python (pyscopg-2-binary).
  
  




  - Create database in Python Console:
   ``` db.createall(); ``` 
 - Connect to database in psql shell.

     ``` /c employee_db1 ``` 

  <h2>Feel free to use the form for any project.</h2>
